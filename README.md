HARDWARE REQUIREMENTS:
* USB Microphone
* Audio plug-in speakers

## Execution Instructions
Installs required packages to run python script
```
virtualenv env 
source env/bin/activate 
sh install.sh
```

After Installation is complete, run python script
`python3 audio.py`
